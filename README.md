# Community

Nowoczesna strona `Community Command Center` dla operacji spolecznosci gier mobilnych live-service.

## Start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run start
```

## Podglad

- Dev: `http://localhost:5173` (lub port podany przy starcie)
- Produkcja Railway: `https://community-web-production.up.railway.app`
